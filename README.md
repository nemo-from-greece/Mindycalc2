A calculator for [Mindustry](https://github.com/Anuken/Mindustry) to facilitate the math when making basically anything. Currently under development, probably to remain that way until 2026 or beyond.

All images are taken from Mindustry and/or the [Unofficial Wiki](https://mindustry-unofficial.fandom.com/wiki/Mindustry_Unofficial_Wiki), and I take no credit for them.

When new content gets added or existing content gets updated, [Data](Data.py) will need to be updated to ensure things function as expected.
